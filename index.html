収集する情報

このページはブラウザおよびネットワークから取得できる技術情報（IPアドレス、ハードウェア情報、ブラウザ環境、フィンガープリント等）を収集し、画面上に表示します。

データの保存について

収集した情報はサーバーへの送信・保存を一切行いません。すべての処理はブラウザ上（クライアントサイド）のみで完結し、ページを閉じると消去されます。サーバー側ログも記録しません。

フィンガープリントについて

Canvas API および Web Audio API を使用したフィンガープリントを生成します。これはブラウザの描画特性・音声処理特性に基づくハッシュ値であり、端末を一意に識別し得る情報です。当サイトではこの値の外部送信・保存は行いません。また、インストール済みフォントの推定にも Canvas API を使用します。

サードパーティサービス

IPアドレス情報の取得に ipinfo.io を使用しています。このリクエストによりあなたのIPアドレスがipinfo.ioのサーバーに送信されます。ipinfo.ioのプライバシーポリシーはこちらをご参照ください。

WebRTC について

ローカルIPアドレスの取得に WebRTC を使用します。RTCPeerConnection を一時的に生成し、ICE candidate からローカルIPを読み取った後、即座に接続を破棄します。外部への接続は行いません。

利用目的

当サイトは個人の技術学習・ポートフォリオを目的として運営されています。収集情報の商業利用・第三者提供は行いません。
