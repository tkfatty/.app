収集する情報

このページはブラウザおよびネットワークから取得できる技術情報（IPアドレス、ハードウェア情報、ブラウザ環境、フィンガープリント等）を収集し、画面上に表示します。

データの保存について

収集した情報はサーバーへの送信・保存を一切行いません。すべての処理はブラウザ上（クライアントサイド）のみで完結し、ページを閃じると消去されます。

フィンガープリントについて

Canvas API および Web Audio API を使用したフィンガープリントを生成します。これはブラウザの描画・音声処理特性に基づくハッシュ値であり、当サイトでは外部送信・保存は行いません。

サードパーティサービス

IPアドレス情報の取得に ipinfo.io を使用しています。このリクエストによりあなたのIPアドレスがipinfo.ioのサーバーに送信されます。

WebRTCについて

ローカルIPアドレスの取得に WebRTC (RTCPeerConnection) を使用します。外部への接続は行いません。
